How I Built My To-Do List App

Published: February 2026

As part of my internship preparation, I was given the task to build a simple but useful project that demonstrates my web development skills. The result was a To-Do List app built with HTML, CSS, and JavaScript.

Why I Chose This Project?

A To-Do List app is a classic beginner project. It allowed me to practice DOM manipulation, event handling, and styling while creating something practical.

Features

  • Add and remove tasks
  • Mark tasks as completed
  • Clean, responsive design

What I Learned

Through this project, I improved my understanding of JavaScript functions, learned how to structure code for readability, styling in a way that is engaging yet fun,and practiced presenting my work professionally on GitHub Pages.
